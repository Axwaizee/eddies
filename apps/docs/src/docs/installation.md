# Overview

Eddies is designed to be used in Next.js with Tailwind CSS, but it works perfectly fine in React and plain CSS.

# Installing `Eddies`

::: code-group

```bash [npm]
$ npm install eddies
```

```bash [yarn]
$ yarn install eddies
```

```bash [pnpm]
$ pnpm install eddies
```
